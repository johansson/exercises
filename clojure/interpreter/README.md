# interpreter

Equation solver in Clojure

## Installation

Install java.
Install lein (leiningen.org).
Clone this repo.

## Usage

    $ java -jar interpreter-0.1.0-standalone.jar program.txt

or

    $ lein run program.txt

## Options

First argument is the file name for the program you want to solve.
